# rl-from-scratch
Collection of classic Environments and Reinforcement Learning algorithms from scratch. Contains Solved Environments and Examples from Reinforcement Learning: An Introduction (2nd Edition) by Richard Sutton and Andrew Barto

- {Car Rental}[environments/car_rental.py]
- {Blackjack}[environments/blackjack.py]
